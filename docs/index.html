How it Works

1. Intelligent Chunking

user-service.js: 1 import bcrypt from 'bcrypt'; 2 import { generateToken } from './utils'; 3 4 class UserService { 5 constructor(db) { 6 this.db = db; 7 } 8 9 ┌─ async authenticate(user, password) { <-- CHUNK 1 10 │ if (!user || !password) return false; 11 │ const hash = await bcrypt.hash(password); 12 │ return this.db.verify(user, hash); 13 └─ } 14 15 ┌─ async resetPassword(email) { <-- CHUNK 2 16 │ const token = generateToken(); 17 │ await this.sendEmail(email, token); 18 │ return token; 19 └─ } 20 }

• Breaks code at meaningful boundaries (functions, classes, docstrings)
• Uses Tree-Sitter parsers for 18+ languages (Python, JS, Go, Rust, etc.)
• Includes DevOps languages (Bash, YAML, Dockerfile, etc.)
• Preserves context and semantic meaning
